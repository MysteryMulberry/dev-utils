# api-test

API测试工具

## 使用方法

```bash
./api-test.sh [参数]
```

## 功能列表

| 参数 | 功能 |
|------|------|
| help | 显示帮助信息 |
| -v | 详细输出模式 |
| -q | 安静模式 |

## 示例

```bash
# 基本用法
./api-test.sh

# 带参数
./api-test.sh --verbose
```

## 安装

```bash
chmod +x api-test.sh
sudo ln -s $(pwd)/api-test.sh /usr/local/bin/api-test
```

## 依赖
- bash
- 常用Linux命令行工具
