# git-helper

Git辅助工具

## 使用方法

```bash
./git-helper.sh [参数]
```

## 功能列表

| 参数 | 功能 |
|------|------|
| help | 显示帮助信息 |
| -v | 详细输出模式 |
| -q | 安静模式 |

## 示例

```bash
# 基本用法
./git-helper.sh

# 带参数
./git-helper.sh --verbose
```

## 安装

```bash
chmod +x git-helper.sh
sudo ln -s $(pwd)/git-helper.sh /usr/local/bin/git-helper
```

## 依赖
- bash
- 常用Linux命令行工具
