# json2yaml

JSON转YAML工具

## 使用方法

```bash
./json2yaml.py [参数]
```

## 功能列表

| 参数 | 功能 |
|------|------|
| help | 显示帮助信息 |
| -v | 详细输出模式 |
| -q | 安静模式 |

## 示例

```bash
# 基本用法
./json2yaml.py

# 带参数
./json2yaml.py --verbose
```

## 安装

```bash
chmod +x json2yaml.py
sudo ln -s $(pwd)/json2yaml.py /usr/local/bin/json2yaml
```

## 依赖
- python3
- 常用Linux命令行工具
