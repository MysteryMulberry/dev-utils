# file-organizer

文件自动分类整理工具

## 安装
```bash
pip install file_organizer
```

## 使用方法
```bash
file-organizer --help
file-organizer <input>
```

## 示例
```bash
echo 'hello' | file-organizer
```

## 选项
- `--verbose`: 详细输出模式
- `--output FILE`: 指定输出文件
- `--help`: 显示帮助信息
