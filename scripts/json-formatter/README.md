# json-formatter

json格式化工具

## 安装
```bash
pip install json_formatter
```

## 使用方法
```bash
json-formatter --help
json-formatter <input>
```

## 示例
```bash
echo 'hello' | json-formatter
```

## 选项
- `--verbose`: 详细输出模式
- `--output FILE`: 指定输出文件
- `--help`: 显示帮助信息
